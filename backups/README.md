# Backups pessoais (Santiago)

Guarda aqui os ficheiros `.json` exportados da app.

**IMPORTANTE:** Esta pasta está no `.gitignore` — os teus dados **não vão para o GitHub**.

## No PC (trabalho)

1. Abre a app → **Settings → Data backup**
2. Clica **Export backup**
3. Guarda o ficheiro nesta pasta, ex.: `santiago-2026-06-12.json`

## Em casa (Firebase)

1. Abre a app → configura Firebase
2. **Settings → Data backup → Import backup**
3. Escolhe o ficheiro desta pasta
4. Cria conta cloud → **Sync now**

Os dados ficam iguais aos do PC.
